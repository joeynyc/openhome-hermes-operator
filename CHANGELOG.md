# Changelog

All notable changes to this project will be documented here.

## 0.1.0 - 2026-05-23

- Scaffolded FastAPI bridge for OpenHome-to-Hermes requests.
- Added fake mode for local demos without hardware or a live Hermes server.
- Added optional bearer-token auth.
- Added OpenHome custom ability folder.
- Added CLI simulator and curl smoke test.
- Added repeatable ability zip packaging.
- Added pytest suite and GitHub Actions CI.
